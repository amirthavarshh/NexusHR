# NexusHR Frontend

NexusHR is a comprehensive Human Resources management system. This repository contains the React frontend for the NexusHR platform.

## Modules
- Employee Hub: Self-service portal for employees.
- HR Module: Advanced tools for HR professionals.
- Manager Module: Team management, attendance, and goal tracking.
- Admin Module: System configuration and dashboard administration.

## Tech Stack
- React
- TypeScript
- Vite
- Tailwind CSS v4
- TanStack Query
- Recharts
- Lucide React

## Getting Started
1. Install dependencies: `npm install`
2. Run the development server: `npm run dev`
3. Build for production: `npm run build`
